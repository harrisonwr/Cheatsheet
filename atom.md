Atom Cheatsheet
===============
Fuck Boy
#### How to open atom? Empty
```
atom
```

#### How to open atom for entire directory?
```
atom .
```

#### How to open atom command line?
```
cmd + shift + p
```

#### How to open markdown preview?
```
cmd + shift + p -> type markdown
```

#### How to find files with shortcut?
```
cmd + p -> type "file name"
```
